# Prompt Engineering Hub

Welcome to the central repository for AI prompt engineering resources within the RuVector project.

## Directory Structure
- `resume_tailor_prompt.md`: Advanced template for ATS-optimized resume and cover letter tailoring.

## Best Practices
1. **Persona Assignment:** Always define a clear role (e.g., "Expert Career Strategist") to set the tone and expertise level.
2. **Context Isolation:** Use clear markers like `<job_description>` or `[INPUT]` to separate user data from instructions.
3. **Outcome Definition:** Explicitly state the required output format (e.g., Markdown, specific sections).
